# Lowcarbon.ai
## Overview

Lowcarbon.ai is a personalised web application designed to encourage users to think more consciously about their AI usage and its environmental impact. Our goal was to show that even small digital habits can contribute to a more sustainable future.

The website asks users a series of questions about their AI usage patterns and then calculates how environmentally harmful their usage may be. Based on the responses, the system provides personalised recommendations to help reduce unnecessary AI usage and promote sustainable digital habits.

## Features
- Personalised AI usage assessment
- Environmental impact percentage calculation
- Sustainable AI usage recommendations
- Simple and user-friendly interface
- Deployed and accessible online

## Technologies Used
### Backend
- Python
- Flask
### Frontend
- HTML
- CSS
### Deployment
- Render

## How It Works
1. Users answer a few questions about their AI usage habits.
2. The system analyses the responses.
3. A percentage score is generated to indicate the environmental impact of the user's AI usage.
4. Personalised suggestions are provided to encourage more sustainable digital behaviour.
   
## Project Goal

The aim of this project is to raise awareness about the environmental impact of AI technologies and encourage users to adopt more responsible and energy-conscious digital habits.
